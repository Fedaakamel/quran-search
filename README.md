# Quran Search App

This is a minimal React (Vite) app for searching the Quran in Arabic with fuzzy matching.

## How to run locally (optional)
1. Install Node.js (v18+ recommended).
2. Run `npm install`.
3. Run `npm run dev`.
4. Open http://localhost:5173

## Deploying to Netlify
Upload this repo to GitHub and connect Netlify to it (build command: `npm run build`, publish: `dist`).
